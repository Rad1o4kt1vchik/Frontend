## 4. ЧТО ВЫ УЗНАЛИ

- Разницу между PropTypes (runtime) и TypeScript (compile-time) валидацией
- Паттерн Controlled Input: почему инпут должен быть привязан к стейту
  через value + onChange, а не читаться напрямую из DOM
- Почему нельзя фильтровать единственный массив в стейте — при удалении
  символов данные теряются; нужен неизменяемый "источник правды"
- Метод .filter() возвращает новый массив, не мутируя исходный —
  это критично для корректного перерендера React
